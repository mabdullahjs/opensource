"Important JavaScript Array Methods"

1. push() Array ke end mein ek ya zyada elements add karta hai. 2. pop() Array ke end se ek element hata deta hai aur usko return karta hai. 3. shift() Array ke start se ek element hata deta hai aur usko return karta hai. 4. unshift() Array ke start mein ek ya zyada elements add karta hai. 5. map() Har element per function apply karke ek naya array return karta hai. 6. filter() Condition ko pass karne wale elements ka naya array banata hai. 7. reduce() Array ke elements ko ek single value mein convert karta hai (jaise sum ya product). 8. forEach() Har element per function apply karta hai, lekin kuch return nahi karta. 9. find() Condition ko pass karne wala pehla element return karta hai. 10. findIndex() Condition ko pass karne wale pehle element ka index return karta hai. 11. includes() Check karta hai ke koi specific element array mein mojood hai ya nahi (true/false return karta h). 12. indexOf() Kisi element ka pehla index return karta hai, agar nahi mila to -1 return hota hai. 13. some() Agar array ka koi bhi ek element condition pass kare to true return hota hai. 14. every() Sirf tab true return karta hai jab har element condition pass kare. 15. sort() Array ke elements ko alphabetical ya numerical order mein sort karta hai. 16. reverse() Array ke order ko ulta (reverse) kar deta hai. 17. concat() Do ya zyada arrays ko merge karke naya array banata hai. 18. slice() Array ke kisi bhi hissa ka naya array return karta hai (original array nahi badalta). 19. splice() Array se elements ko add ya remove karne ke liye use hota hai (original array modify hota hai). 20. join() Array ke elements ko ek string mein convert karta hai, aur separator specify kiya ja sakta hai.
